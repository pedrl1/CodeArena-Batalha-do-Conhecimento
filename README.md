## About the project

Code arena is a game made entirely in Java 8, specifically made for the final project of "Object Oriented Programming Languages" class at Pernambuco's State University Polytechnical School, also known as *POLI - UPE*

The project itself was made using all the characteristics of the paradigm, such as:

- **Polymorphism**
- **Class inheritance**
- **Classes**
- **Objects** (duh)
- **Encapsulation**
- **Methods**
